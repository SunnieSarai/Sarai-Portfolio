## 🧰 Tech Stack 
| Tool/Library | What It Does | Why I Chose It |
|--------------|---------------|----------------|
| **HTML5** | Structure & semantic markup | Industry standard; I wanted to practice what i've learned |
| **CSS3**  | Animations, layout, responsive design | Full creative control without framework bloat; custom keyframe animations. I was able to teach myself something new |
| **JavaScript (Vanilla)** | Interactivity, DOM manipulation, API calls | No framework overhead; lighter bundle, faster load times, and complete control over UX |
| **TailwindCSS** | Utility-first CSS framework | Speeds up responsive design; consistent spacing/colors |
| **Lucide Icons** | Lightweight SVG icon library | Modern, consistent, and accessible icon set (replaces FontAwesome) |
| **Formspree** | Backendless form handling | Free tier; sends form submissions to my email without needing a server |
| **Intersection Observer API** | Scroll-triggered animations | Native browser API; more performant than scroll-event listeners |
| **Element SDK (custom)** | On-page content/color editing | Allows dynamic content changes without touching code — useful for portfolio iterations |

## ✨ Features

- **Fully responsive** — works on mobile, tablet, and desktop
- **Smooth scroll animations** powered by Intersection Observer
- **Dynamic typewriter effect** on hero tagline
- **Live counter animation** for project stats
- **Project filtering** by category (Code / Writing / Research) — ready to expand
- **Contact form** integrated with Formspree (no backend required)
- **Mobile-friendly navigation** with hamburger menu
- **Geometric floating shapes** for visual depth
- **Configurable via Element SDK** — colors, fonts, and text can be updated dynamically
