🌸 Featured Projects
1. Bundle Impact Visualizer (BIV)
Tech Stack: TypeScript · HTML/CSS · MongoDB · Vite · cli-table3 · npm

A CLI tool that visualizes npm package bundle sizes and dependency impact directly in your terminal. Built for developers who want to understand exactly what they're installing.

What I built:

Command-line interface that fetches and analyzes npm package metadata

Terminal-based table visualizations using cli-table3

Dependency tree traversal to calculate total bundle impact

MongoDB storage for caching package data and improving performance

Why it matters: Most developers don't know how many kilobytes they add with each npm install. BIV makes that visible — right in your terminal, before you commit.

Repository: github.com/sunniesarai/BIV (update with actual repo link)
Live Demo: biv-murex.vercel.app

2. Canopi — Goal Nesting Task Manager
Tech Stack: React · TypeScript · Node.js · Express · MongoDB · Vite · Tailwind CSS

A task management tool built to combat functional paralysis. Canopi breaks large goals into small, blooming steps using a botanical metaphor: Garden → Plot → Seed → Task.

What I built:

Nested MongoDB schema (Gardens contain Plots, Plots contain Seeds, Seeds contain Tasks)

Full CRUD operations across all four levels of data

RESTful API endpoints following hierarchical URL patterns

Progress bars and completion tracking at the seed level

CSS bloom animations when all tasks in a seed are complete

The story: I built Canopi for people (like me) who freeze when a goal feels too big. By nesting goals, each step becomes manageable. When you complete every task in a seed, it blooms into a flower — a tiny reward for progress.

Repository: github.com/sunniesarai/Canopi
Live Demo: (deployment in progress — coming soon)

3. Pantry-Pal
Tech Stack: React · Express · RESTful APIs · Jest · MongoDB

A virtual pantry with expiration date tracking and alerts. Built to help households reduce food waste and save money.

What I built:

Full-stack CRUD application for pantry item management

Expiration date tracking with automated alerts

RESTful API design for item creation, update, deletion

Unit testing with Jest for backend routes

Responsive React frontend

Why it matters: The average household wastes 30% of the food it buys. Pantry-Pal turns that problem into a software solution — track what you have, get alerts before things expire, and stop throwing away money.

Repository: github.com/BloodR00t/pantry-pal
