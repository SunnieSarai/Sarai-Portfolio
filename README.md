## 📌 Overview

This is my personal portfolio — a visual and interactive space where I showcase my work as a software engineer who sits at the intersection of technology, culture, and narrative. The site reflects my background in sociology and African Studies, translating that lens into clean code and intentional design.

Visitors can explore my projects, learn about my approach, and reach out for collaborations.

## ✨ Features

- **Fully responsive** — works on mobile, tablet, and desktop
- **Smooth scroll animations** powered by Intersection Observer
- **Dynamic typewriter effect** on hero tagline
- **Live counter animation** for project stats
- **Project filtering** by category (Code / Writing / Research) — ready to expand
- **Contact form** integrated with Formspree (no backend required)
- **Mobile-friendly navigation** with hamburger menu
- **Geometric floating shapes** for visual depth
- **Configurable via Element SDK** — colors, fonts, and text can be updated dynamically
